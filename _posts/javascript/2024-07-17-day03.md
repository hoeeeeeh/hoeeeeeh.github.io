1. 데이터 저장 자료구조 설정

```javascript
prolog: Map;
// XML 메타 데이터 기록 (Ex. { "version" : "1.0" , "encoding" : "utf-8" })

elements: Array;
// 최상위 태그 요소부터 위에서 아래로 순서대로 태그 단위로 기록

element: Map;
// element: String (태그 이름?)
// attribues: Array<attribute>
//태그의 모든 속성을 attribute 구조로 저장

attribues: Map;
// name: String (속성 이름)
// value: String?Number (속성 값)

<!DOCTYPE html> -> (주석 Comment : <!DOCTYPE> 처럼 앞에 !로 시작하면 XML 주석으로 처리하고 저장하지 않는다.)
<HTML lang="ko"> -> element {

    attributes = [attribute{
        name: "lang",
        value: "ko"
    }]
}
```
